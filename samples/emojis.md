## Common in Commits & PRs :tada:

These emojis are often used in commit messages and pull requests:

- :sparkles: `:sparkles:` — New feature
- :bug: `:bug:` — Bug fix
- :memo: `:memo:` — Documentation
- :lipstick: `:lipstick:` — UI/style updates
- :rocket: `:rocket:` — Performance improvements
- :white_check_mark: `:white_check_mark:` — Adding tests
- :lock: `:lock:` — Security fix
- :arrow_up: `:arrow_up:` — Upgrading dependencies
- :arrow_down: `:arrow_down:` — Downgrading dependencies
- :recycle: `:recycle:` — Refactoring code
- :fire: `:fire:` — Removing code/files
- :construction: `:construction:` — Work in progress
- :rotating_light: `:rotating_light:` — Fixing linter warnings
- :green_heart: `:green_heart:` — Fixing CI build
- :tada: `:tada:` — Initial commit / celebration
- :bookmark: `:bookmark:` — Release / version tags
- :hammer: `:hammer:` — Tooling changes
- :truck: `:truck:` — Moving/renaming files
- :boom: `:boom:` — Breaking changes
- :art: `:art:` — Improving structure/format
